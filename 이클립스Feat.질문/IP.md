### IP?(internet protocol)

 => 인터넷을 접속하기 위한 주소!
  (주소 변경이 가능하다)
  
 ###  DB접속
 
    1. IP 
    
     내가 DB에 접속하려면
     DB가 설치된 컴퓨터 IP를 알아야 함!
     cmd - ipconfig - IPv4
     192.168.0.66 끝번호만 
     192.168.0.56 == localhost == 127.0.0.1 (같은 말)

    2. PORT (방)
    
     3306  
    192.168.0.56:3306  
    192.168.0.56 주소를 가진 컴퓨터 3306 방에 들어가겠다.

    3. (option) database 이름
    
    데이터베이스 -> 테이블 -> 데이터
    프로젝트 -> 패키지 -> 클래스

    
### IP가 두 종류 있음
